JavaLearn Notes - 2

Posted on 2020-02-15 Edited on 2021-01-12

基本数据类型

  1. C/C++用的ASCII码(8位)，Java用的Unicode码(16位)
  2. true不是1，false不是0！这是两种不同类型

变量

  1. 变量使用之前必须初始化值！

参数传递

按值传递

  • Java中只有按值传递！！！
  • 对象作为参数传入的是地址，因此内容可以被更改，但仍是按值传递

不定长参数

  • 相当于传了一个数组进来
  • 不定长参数要放在所有参数的最后面

方法重载

  • 方法名称相同，参数列表不同
  • 编译器根据方法签名决定调用哪个方法
  • 注意
    • 有时调用方法会有两个或更多可能的匹配，编译器无法判断哪个更为精确，这称为歧义调用（ambiguous invocation）
    • 被重载的方法必须具有不同的参数列表，不能基于不同修饰符或返回值类型来重载方法

      类

  • public
    • 公共类，可以被其他类所使用
    • declares that the class can be used by an class regardless of its package
    • 无任何限制
    • 一个文件只能有一个public类
  • 无修饰/默认说明
    • a class can be used only by other classes in the same package
    • 仅仅能在同一个包中的其他类引用
  • abstract
    • declares that the class cannot be instantiated
    • 宣布该类不能被实例化
  • final
    • declares that the class cannot be subclassed
    • 宣布该类不能有子类

成员变量的访问权限

  • public
    • 公共访问控制符
    • 被所有类访问
  • 默认访问控制符
    • 被同一包中的其他类访问
  • private
    • 私有访问控制符
    • 被该类自身访问
  • protected
    • 保护访问控制符
    • 该类自身、同一个包中的其他类、其他包中的子类访问
  • private protected
    • 私有保护访问控制符
    • 该类自身、所有子类访问

String类

“+”连接符

  • “+”连接符对于直接相加的字符串常量效率很高，因为在编译期间便确定了它的值，形如”I”+”Love”+”Java”; 在编译期间便被优化成了”ILoveJava”
  • 对于间接相加（包含字符串引用，且编译期间无法确定值的），形如”a” + s1; 效率要比直接相加低

代码解释

  1. public 关键字 公共的类 后面要有一个空格 一个文件中只能有一个public类！
  2. class 关键字 类 后面要有一个空格
  3. Test1 类名称 命名规范 驼峰命名法 首字母大写 单词分隔首字母大写
  4. {} 配对 类体 class body
  5. public 关键字 公共方法
  6. static 静态方法 静态方法可以直接访问 不需要实例化
  7. void 方法的返回值为空
  8. main 主方法 入口方法 程序从这个方法进入 并开始执行
  9. main方法所在类是主类，主类必须跟文件名相同
  10. (String[] args) main方法的参数 类型是字符串数组类型
  11. {} 方法体 method body
  12. System 类型里面的一个类 JDK自带的一个类
  13. out 静态变量
  14. println 方法 out变量里面的一个方法 输出到控制台 ln 换行输出
  15. “Hello Java! Here we meet again!” 字符串参数 输出的内容是它
  • 关于println
    • 会把括号里的内容传换成字符串输出，输出对象会自动调用对象的toString()方法
    • 注意下面两句话输出的区别
  1. ; 表示语句的结尾
  2. 注意区分大小写（大小写敏感）（类名、文件名、关键字等）

eclipse常用快捷键

  • Alt+/ 自动补全
  • Ctrl+/ 注释一行
  • Ctrl+Shift+/ 添加块注释
  • Ctrl+1 修复错误
  • Ctrl+Shift+F 格式化代码
  • Ctrl+Shift+O 自动导入包
  • F11 Ctrl+F11 调试 直接运行
  • F5 F6 F7 F8 调试常用键

标识符的命名规则

  • 标识符只能由字母、数字和下划线和美元符号（$）组成
  • 所有的标识符都应该以字母（A-Z或者a-z），美元符（$）、或者下划线（_）开始
  • 关键字不能用作标识符
  • 标识符是大小写敏感的
  • 合法标识符举例：age、$salary、_value、__1_value
  • 非法标识符举例：123abc、-salary
