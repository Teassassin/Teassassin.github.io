δ邻域的几何意义

$ U\left(P_{0}, \delta\right) $ 表示 $ x O y $ 平面上以点 $ P_{0}\left(x_{0}, y_{0}\right) $ 为中心, $ \delta>0 $ 为半径的圆的内部的点$ P(x, y) $ 的全体

邻域与区间（区域）

邻域通常表示“一个局部位置”，比如“点x0的δ邻域”，就可以称为“点x0的附近”。这个“附近”到底多远，既难以说明也没有必要说明。

2. 函数极限的定义

ε-δ语言：

$ \lim\limits_{x\rightarrow\infty}f(x)=A\Leftrightarrow\forall\varepsilon>0,\exist X>0, 当|x|>X时，有|f(x) - A|<\varepsilon $

左右极限都包括

函数极限存在的充要条件

$ \lim {x \rightarrow x{0}} f(x)=A \Leftrightarrow \lim {x \rightarrow x{0}^{-}} f(x)=A, $ 且 $ \lim {x \rightarrow x{0}^{+}} f(x)=A $

或

$ \lim {x \rightarrow x{0}} f(x)=A \Leftrightarrow f(x)=A+\alpha(x), \lim {x \rightarrow x{0}} \alpha(x)=0 $（脱帽法。这里不代表$f(x)=A$，之间有误差$α(x)$）

5. 夹逼准则

$ \lim [g(x)-\varphi(x)] $ 存在并不能说明 $ \lim g(x), \lim \varphi(x) $ 都存在，从而也不能保证 $ \lim f(x) $ 存在.

6. 洛必达法则

  1. $\frac{0}{0}$**或$\frac{\infin}{\infin}$**

  2. 分子分母都可导

  3. $ \lim _{x \rightarrow a} \frac{f^{\prime}(x)}{F^{\prime}(x)}\left(\right. $ 或 $ \left.\lim _{x \rightarrow \infty} \frac{f^{\prime}(x)}{F^{\prime}(x)}\right) $ 存在或无穷大

则$$\lim _ { x \rightarrow a } \frac { f ( x ) } { F ( x ) } = \lim _ { x \rightarrow a } \frac { f ^ { \prime } ( x ) } { F ^ { \prime } ( x ) } ( \lim _ { x \rightarrow \infty } \frac { f ( x ) } { F ( x ) } = \lim _ { x \rightarrow \infty } \frac { f ^ { \prime } ( x ) } { F ^ { \prime } ( x ) } )$$

右存在，则左存在；但左存在，并不意味着右一定存在
